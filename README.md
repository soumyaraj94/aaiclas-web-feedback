# AAICLAS Cargo Agents Feedback Form

A professional feedback collection system for AAICLAS Cargo Agents with Flask backend and Google Sheets integration.

![Flask](https://img.shields.io/badge/Flask-3.0.2-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple)
![Python](https://img.shields.io/badge/Python-3.8+-yellow)

## 🚀 Features

- ✅ Mobile-responsive design
- ✅ Client & server-side validation
- ✅ Google Sheets storage
- ✅ 10-digit mobile validation
- ✅ Personalized success message with 3-second countdown
- ✅ Submit & Clear buttons
- ✅ Demo mode (works without Google Sheets)

## 📋 Form Fields

| Field | Required | Validation |
|-------|----------|------------|
| Agent Name | ✓ | Min 2 characters |
| Company Name | ✓ | Min 2 characters |
| Mobile Number | ✓ | 10 digits (6-9) |
| Email Address | ✓ | Valid format |
| Service Rating | ✓ | 1-5 stars |
| Your Feedback | ✓ | Min 10 characters |
| Remarks | ✗ | Optional |

## 🛠️ Tech Stack

- **Backend:** Python Flask
- **Frontend:** Bootstrap 5, HTML/CSS/JS
- **Database:** Google Sheets API
- **Deployment:** Render / Heroku
